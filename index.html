How to play

Welcome to the quiz!
Below you will be able to choose a difficulty and question category.
Once you have pressed a category you will be taken to a page which has relevant questions.
When you get a question right it will display green, when it is wrong it will go red and
highlight the correct answer in green, there is no time limit so have fun!


This is my quiz website I have designed as a portfolio piece to demonstrate my web-development ability.


Below you will find 25 categories of questions which you can choose from to
do a 10 question quiz. The quiz questions are genreated from an API which can
be found at opentdb.
